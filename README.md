# Netflix Clone

## Description
This project is a frontend clone of the Netflix homepage, developed using HTML and CSS. It replicates the visual layout and basic user interface of Netflix, focusing on design and responsiveness.

## Features
- Responsive layout for different screen sizes
- Structured homepage with sections similar to Netflix
- Clean and organized UI design

## Technologies Used
- HTML
- CSS

## Project Structure
- index.html: Main HTML file
- style.css: Styling for the application
- Pics/: Contains all images used in the project

## How to Run
Download or clone the repository and open the index.html file in any web browser.
